Introduction:
You will create an input loop to accept user input from the command line. You will count the number of keystrokes, alphabetical characters, numbers, and vowels.

The Main Function:
The main function will execute the input loop.

The Input Loop:
Create an input array with a large enough size to hold a sentence. Exit the loop when the user inputs the SENTINEL character of '\n' (newline) only. When the user enters a sentence, send the input array to the keystrokes, alphabetical, number, and vowel functions.

Use the fgets() function to collect a whole line of user input into your input character array.
     https:// en.cppreference.com/w/c/io/fgets
For the last parameter, use stdin.

You may find this reference to be useful for any string and character manipulations along with any testing functions that you'll need:
     https:// en.cppreference.com/w/c/string/byte

When the user is entering input, please print out instructions to the user on each input loop iteration. You should tell the user to input a sentence and how to exit the program using the sentinel character.

The Keystrokes Function:
Count the characters, not including the null-terminator nor the newline characters: please do count space characters and punctuation. Return an int from this function.

The Alpha Function:
Count the alphabetical characters. Return an int from this function.

The Numeric Function:
Count the numeric characters. Return an int from this function.

The Vowels Function:
Count the vowels in the sentence. Return the count as an int.

Showing the Data in the Input Loop Function:
Please format the output like this:

Alpha:        5
Numeric:      2
Vowels:       5

Use printf()'s padding functionality for formatting.
