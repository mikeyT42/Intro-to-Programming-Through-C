Introduction:


This program will be practice with using enumerations, arrays, strings, and structures. This will also be the only program where I will provide a function. We haven't been introduced to logical statements, so that is why I am providing it.

You will be writing a program that is going to be a "Library."  You will have some books that you will define, and print them all out.

genre Enumeration:

This will be an enum that you will define that will represent book genres. Here are the values:

FICTION
NON_FICTION
SCIENCE_FICTION
MYSTERY
BIOGRAPHY

These values will then be used to print the kinds of genres.

book Structure:

Make a constant using #define. I want the name to be STRING_LEN with a value of 100.

Now onto the book structure. We need a string called title with a length of STRING_LEN. Use this constant in the length part of the declaration. Then declare a string called author with the same length. Declare another data member with a type of unsigned int called year. Then declare another data member with a type of genre called genre.

void print_book(book book):

This function will be solely responsible for printing its given book argument. Make the argument const too. I want the printed output to look like the below:

book {
  title = The War of the Worlds
  author = H. G. Wells
  year = 1898
  genre = Science Fiction
}

It is possible to use a single printf statement to do this, but you can also do this in about 6 printf statements. Be sure to make use of the given genre_to_string(const genre genre) function.

main:

After the salutation, declare and initialize a book array with 6 books. Initialize it with any 6 books that you like. Then, printf this message:

Our library has %lu book(s).

Be sure to put 2 newlines at the end. Calculate the length of the array in this printf statement. Then call the printf_book function for each book in our library array.

Conclusion:

Be sure to follow our formatting rules. Between function definitions put our function separator comment. Also, remember our variables and function names are following snake case. Be sure that indenting is proper as well. 4 spaces, and no more than 80 characters wide.
