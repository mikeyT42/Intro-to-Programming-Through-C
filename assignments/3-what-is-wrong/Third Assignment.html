This assignment doesn't really have a description: there isn't anything to describe. This program is fully written, but is just riddled with syntax issues and various bugs. It is your job to fix it.

This program will heavily flex your debugging muscles. It will also act to demonstrate the various data type sizes and certain concepts that we have covered so far. Everything can be found in previous assignments.

Here are some tips to help you with this program:
  • Take every problem one step at a time.
  • Start with the compilation errors at the top of the file, and then work your way down.
  • "Comment out" certain pieces of code for a few variable assignments.
  • "Comment out" individual if statements to get the answers to data sizes.
    • Take them one at a time.
    • These answers should then be used as the values in the corresponding variables.
  • Make sure every printf statement is able to execute.
  • Utilize all of your available tools.
  • Make sure there are no warnings by the end.
