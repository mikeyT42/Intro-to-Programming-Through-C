Introduction:

This program is going to be our second foray into pointer practice. This program will do some simple mathematics: we will create some sums and averages. We will gather some numbers from the user.

Structures:

This program will be our first one to utilize struct definitions. Our first typedef of a struct will be sums. It will have 3 data members: a double variable called positive; a double variable called negative; and a double variable called overall. The second typedef of a struct will be called counts. This has 3 data members as well: an unsigned, int variable called positive; an unsigned, int variable called negative; and an unsigned, int variable called overall. Finally, make a typedef of a struct called averages. It will have 3 data members: a double variable called positive; a double variable called negative; and a double variable called overall.

main:

This will behave exactly like our other main methods: print banners and execute an input_loop().

input_loop:

This will return a boolean value, or enum if you'd like, of true or false: true representing continue the input loop and false indicating to stop the program. This function will read the user's input into a string: make sure it is pretty big; 256 characters should be sufficiently sized. Indicate to the user that floating point or integers are acceptable; also indicate to only enter up to 10 numbers. The newline character will continue to be our sentinel.

First check for the sentinel: exit if it has been entered. Then, utilize sscanf to read our string into a double array: this array will hold up to 10 numbers. So, some context on sscanf: this function reads a string into any number of variables on the pattern provided, kind of like how scanf reads from the standard input stream; or how printf outputs data. The first argument is a string, the second argument is the full pattern to match on--in this case all 10 floating point numbers--and the last arguments will be the address of each array element we can possibly save into. The return value of sscanf is the number of matches it had from the string. So, if the user input 5 things, the return value would be 5.

This time around we won't have a validate function. Simply validate the number of matches. If it fails to validate, return true.

Then declare and initialize our 3 structs: each member of the structures should be initialized to 0.

We will then execute sum_and_count(), average(), and print_table() functions before returning true.

sum_and_count(double numbers[], unsigned int numbers_len, sums *sums, counts *counts):

First off, make each argument as immutable as possible. Iterate over the numbers array. For each element in the array, check if it is positive (greater than or equal to 0) or negative. If it is positive, then add to the positive member of sums and increment the positive member of counts. Do the same for negative numbers. For every array iteration regardless of positivity or negativity, add to sums overall and increment counts overall.

average(sums sums, counts counts, averages *averages):

First off, make each argument as immutable as possible. Calculate the averages for both positive, negative and overall numbers. Be sure to check first if the counts for each respective kind is 0; if you don't do this you could get a divide by 0 bug.
