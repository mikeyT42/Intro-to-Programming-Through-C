Introduction:

This fifth assignment is going to be a chance to work with pointers. We will be pointing to local variables, and then sending them to a function to do calculations with in order to have a "multi-return."

This program will be a change calculator. It will tell the user how many quarters, dimes, nickels, and pennies it takes to make their provided number. You will accept string input, though we will do things a bit differently. Also, put a comment at the bottom of the program of your output.

main:

You will output a banner and a salutation. You will have an input loop function that returns true or false on whether or not to exit the loop or not. The function it will call until false will be called input_loop(void). This will be inside of a loop defined in the main function. Use the do-while loop again. Also, import stdbool.h so that you get access to the boolean type. Use the type and the provided boolean values inside of stdbool.h.

input_loop(void):

This function will return a boolean value of true or false when the user wants to exit: the sentinel value will be -1. This function will be the one to gather user input. After gathering, it will check for the sentinel, validate, calculate the change, and then print out the values. If the user wants to exit, do not validate, calculate, or print out the values. If the user wants to exit, return false; if the user wants to continue validate, calculate, and then print; return true in this case.

We will be reading data differently this time around. We will be using scanf(). Read up on scanf(). Please use the double format to read a double value. Read it not into a string, but a double variable. If the user enters -1, that is when you exit. The prompt for input should tell the user to input a number, and for it to be between 0 and 1.

Once you have calculated the change, use printf() to format a string like this:
    The amount you gave was $0.50, your change is 2 Quarters, 0 Dimes,
    0 Nickels, and 0 Pennies.

validate(const double input_cost):

Simply check if the user did not enter a number between 1 and 0. If not, return false, if the user did then input true.

calculate_change(double *input_cost, int *num_quarters, int *num_dimes, int *num_nickels, int *num_pennies):

I have omitted const from these parameters. Please make the parameters as immutable as possible. Please make a local variable to store the cost in cents. You'll have to multiply the input cost accordingly for this variable. Divide this by 25 to get the number of quarters. Then, use the modulus operator to save into the change that is left. Then divide that by dimes, do the modulus, etc. After the last modulus, all that there is left is the penny count.
